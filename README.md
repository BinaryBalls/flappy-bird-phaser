# 🐦 Flappy Bird Clone

A Flappy Bird-style game built using Phaser.js.

## 🚀 Features
- 🎮 Responsive full-screen gameplay
- 📈 Dynamic difficulty scaling
- 🔊 Procedural sound effects (Web Audio API)
- 🏆 High score tracking (localStorage)
- 🌙 Night mode toggle
- 🎯 Smooth animations & controls

## 🕹️ Controls
- SPACE / UP Arrow / Click → Flap
- SPACE → Restart after crash
